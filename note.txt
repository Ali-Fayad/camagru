Database Tables

        User

                id (auto-generated)

                username (letters and digits only, saved in lowercase)

                email (must be valid, contain @)

                password (hashed)

                enableNotification (boolean, default true; if true, email notifications are sent)

                isVerified (boolean, default false)

                verification_code (for email verification)

                created_at (auto-generated)

                updated_at (auto-generated)

        Images

                id (auto-generated)

                userId (foreign key to users.id)

                photo (stored as file path)

                caption (text)

                likesCount (auto-generated)

                commentsCount (auto-generated)

                created_at (auto-generated)

                updated_at (optional, for editing captions)

        Likes

                id (auto-generated)

                userId (who liked the image)

                imageId (the image that was liked)

                created_at (auto-generated)

                Unique constraint on (userId, imageId) to prevent duplicate likes

        Comments

                id (auto-generated)

                userId (who wrote the comment)

                imageId (the commented image)

                description (text of the comment)

                created_at (auto-generated)

API Endpoints

        Verification

                mailer/

                    Body: { email: string, code: number }

                mailer/verify

                    Body: { email: string }

                    Returns user verification status

        Notifications

                notf/

                    Body: { imageId: number, userId: string }

                    Returns notification info for the user

        User Account

                user/login/

                    Body: { username: string, password: string }

                user/signup/

                    Body: { email: string, username: string, password: string }

                    Returns success message; must verify email before full access

                user/delete/

                    Body: { userId: number }

                user/update/

                    Body: { email: string, username: string, password: string, enableNotification: boolean }

        Posts / Events

                posts/get/

                    Returns last 5 posts (support for pagination recommended)

                posts/add/

                    Body: { userId: number, image: string, caption: string }

                like/

                    Body: { userId: string, imageId: number }

                comment/

                    Body: { userId: string, imageId: number, comment: string }
